# PC2T_project - Systém pro správu zaměstnanců
- Vypracoval:     Marek Focher (269638)
#
- Repozitář obsahuje konzolovou aplikaci v jazyce Java určenou k evidenci a celkové správě zaměstnanců.
- Aplikace například umožňuje přidávání zaměstnanců, organizaci do skupin, evidenci spoluprací a nebo generování statistik.
- Podporuje ukládání a načítání dat pomocí SQL databáze a lokálních souborů.
